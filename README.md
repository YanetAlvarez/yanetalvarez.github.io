# Supplementary Material

Material suplementario para el paper: animaciones (.gif) y vídeos (.mp4).

Estructura:
- index.html — página para mostrar GIF y MP4 embebidos.
- media/ — carpeta local para los archivos (opcional si embeber desde Drive).
- .gitattributes — configuración para Git LFS.
- README.md — esto.
- .nojekyll — evita que GitHub Pages ignore archivos que comiencen con _.

Notas:
- Repositorio inicialmente privado. Invita colaboradores por su usuario de GitHub o por correo.
- Si los MP4 >100 MB utiliza Git LFS para subirlos al repo.
- Alternativa: mantener los binarios en Google Drive / Zenodo / OSF / S3 y embeberlos desde allí (recomendado para archivos grandes).
